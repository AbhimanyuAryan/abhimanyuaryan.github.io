I served as **Technical Reviewer** for this book, which covers building full-stack web applications with the Julia programming language using the Genie web framework.

The book walks through creating reactive dashboards with Stipple.jl, building REST APIs with Genie, and working with databases via SearchLight.jl — all core packages I contributed to as part of the GenieFramework ecosystem.
