A clone of **Google Tilt Brush** built with **Apple ARKit** and **Unity**. Google Tilt Brush is an amazing example of what Augmented Reality can offer — a tool where you paint in a 3D world using AR. This project replicates that experience as a learning exercise in ARKit.

The app allows users to draw and paint in 3D space using their device's camera, with strokes anchored to the real world via ARKit's spatial tracking.

### Tech Stack
- **Apple ARKit**
- **Unity**
- **C#**
